== How to Use ==

1. Visit [ReviewManagementTool.com](https://reviewmanagementtool.com)
2. Create your widget by adding testimonials (text, video, or audio)
3. Tag them and generate a widget
4. Copy the Widget ID
5. In WordPress:
   - Use shortcode:  
     `[rmtool widget_id="YOUR_ID" type="js"]` or `[rmtool widget_id="YOUR_ID" type="iframe"]`  
   - Or insert the **Review Management Widget** block in the editor and paste the widget ID
